# Mobile

Bu klasor Flutter tabanli mobil istemci iskeletini icerir.

Calistirma:

```bash
cd mobile
flutter pub get
flutter run
```

Not:
- Bu ortamda `flutter` kurulu olmadigi icin burada build dogrulamasi yapilamadi.
- Android/iOS klasorleri gerekirse `flutter create .` komutuyla tamamlanabilir.
